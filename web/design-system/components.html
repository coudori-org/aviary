Buttons

Three variants × two sizes, plus icon-only. Flat fills only — no gradients, no elevation on the base state.

shadcn mapping

.btn-primary → <Button variant="default">
.btn-outline → <Button variant="outline">
.btn-ghost → <Button variant="ghost">
.btn-sm → size="sm"
.btn-icon → size="icon"
Heights: default 30px, sm 26px, icon 30×30. Radius: 7px default, 6px sm. Weight: 500. Transition: 120ms on bg+border. Never animate color.

Inputs

Text field, textarea, select — all share the same shell height and 7px radius. Focus = accent soft fill + accent border.

shadcn mapping

.input → <Input />
.input[textarea] → <Textarea />
.input[select] → <Select />
Height: 30px. Focus: fills to --bg-raised and border becomes --accent-blue-border. No outline, no glow. Pair with <Label> above.

Status Dots

Tiny colored dots with soft halo. Used in run lists, session rows, workflow nodes, nav counts.

shadcn mapping

.dot-live → <StatusDot variant="live"/>
.dot-warn → variant="warn"
.dot-error → variant="error"
.dot-idle → variant="idle"
Live dot has a soft halo (box-shadow: 0 0 0 3px token-soft). Add pulse prop for slow 1.6s breathing on "running" state — reduced-motion auto-disables.
